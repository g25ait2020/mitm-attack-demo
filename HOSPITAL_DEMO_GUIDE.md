# 🏥 Hospital Patient Portal - MITM Attack Demo Guide

## Overview

This guide explains how to use the Hospital Patient Portal demo to understand how Man-in-the-Middle (MITM) attacks can compromise sensitive medical data.



## Quick Start

### One-Click Demo 

```bash
run_hospital_demo.bat
```

This automatically starts:
1. Hospital Server (port 8080)
2. MITM Attacker Proxy (port 8081)
3. Hospital Client GUI
